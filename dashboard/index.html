Tool Guide

End-user flow: upload content, save memory, query memory, verify privacy, and reset data.

1) Upload Source File

Use the Upload & Index section. File will be copied to my_data/ and indexed for retrieval.

2) Enable Tools In Chat

In Archestra Chat, verify save_memory and query_personal_memory are enabled.

3) Save User Memory

Ask the agent to call save_memory for user facts like preferences, goals, and profile details.

4) Query Memory Safely

Ask the agent to call query_personal_memory. Sensitive content is automatically redacted.

5) Observe Telemetry

Use Live Events, Redaction Visualizer, and Indexed Files to audit behavior in real time.

6) Delete / Reset

Remove one file from Indexed Files or use Clear All Uploaded Data for full cleanup.

Save Memory Prompt

Call tool save_memory with {"fact":"I like black coffee."} and reply only: SAVED

Query Memory Prompt

Call tool query_personal_memory with {"topic":"what do I like?"} and return only tool result.
